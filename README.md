# КТЖ SPORT — приложение с регистрацией и Supabase

## Что уже работает

В проекте есть:

- регистрация по любой электронной почте;
- вход по email и паролю;
- подтверждение email через Supabase;
- профиль участника;
- список событий;
- заявка на соревнование;
- раздел «Мои заявки»;
- Row Level Security: пользователь видит только свои данные;
- демо-режим без базы данных.

## Быстрый просмотр без Supabase

1. Откройте `index.html`.
2. Создайте демо-аккаунт.
3. Заполните профиль.
4. Подайте заявку.

Данные сохраняются только в localStorage текущего браузера.

## Подключение настоящей базы Supabase

### 1. Создайте проект

Откройте Supabase и создайте новый проект.

### 2. Создайте таблицы

Откройте:

`SQL Editor → New query`

Скопируйте содержимое файла `supabase-schema.sql` и нажмите **Run**.

### 3. Вставьте ключи

В Supabase откройте:

`Project Settings → API`

Скопируйте:

- Project URL;
- Publishable key или anon public key.

Откройте `config.js`:

```js
window.KTZ_CONFIG = {
  supabaseUrl: "ВАШ_PROJECT_URL",
  supabaseAnonKey: "ВАШ_PUBLIC_KEY",
  demoMode: false
};
```

Никогда не вставляйте в сайт `service_role` key.

### 4. Настройте адрес сайта

В Supabase:

`Authentication → URL Configuration`

Укажите:

- Site URL;
- Redirect URLs для вашего домена и локального адреса.

### 5. Подтверждение электронной почты

В Supabase:

`Authentication → Providers → Email`

Оставьте включенным подтверждение email для официального проекта.

## Добавленные разделы в версии 2

### Спортивные инструкторы

Список находится в начале файла `app.js`:

```js
const INSTRUCTORS = [
  { region: "Астана", name: "ФИО", phone: "+7...", email: "..." }
];
```

Замените данные-заглушки на реальные ФИО и контакты.

### Фото каждого события

Ссылки находятся в `config.js`:

```js
eventPhotoUrls: {
  "futsal-2027": "ССЫЛКА_НА_ФОТО",
  "volleyball-2027": "ССЫЛКА_НА_ФОТО",
  "summer-games-2027": "ССЫЛКА_НА_ФОТО"
}
```

Можно использовать ссылки на OneDrive, Google Drive, Яндекс Диск или корпоративный фотоархив.

### Результаты

Возвращён раздел результатов IX и X Спартакиад АО «Самрук-Қазына».
